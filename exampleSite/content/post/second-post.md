Paper Reader keeps typography simple:

- comfortable line height
- readable article width
- visible links
- automatic light and dark mode

Inline code such as `hugo server` uses a monospace font. Code blocks are intentionally plain:

```toml
theme = "paper-reader"

[pagination]
  pagerSize = 10
```
